# 🏆 World Cup & Tournament Friends Betting App

אפליקציית ווב קלילה ומודרנית לניהול טורניר הימורים וניחושים בין חברים (מונדיאל, יורו, ליגת האלופות ועוד). האפליקציה מאפשרת למשתתפים להזין ניחושים, לעקוב אחר טבלת הלידרבורד, ולמנהל הטורניר לנהל כספים, לאשר תשלומים ולנעול את הטורניר עם שריקת הפתיחה.

## 🚀 פיצ'רים מרכזיים
- **לובי קבוצות חכם:** זיהוי משתמש אוטומטי מבוסס `localStorage` ומעקב אחר הקבוצות אליהן הוא רשום.
- **מערכת הימורים דינמית:** ממשק הזנת ניחושים (אלופה, סגנית, מלך השערים) עם חסימה אוטומטית ברגע שהטורניר ננעל.
- **פאנל ניהול ותשלומים (Admin):** שליטה מלאה למנהל הקבוצה - מעקב מי שילם (כולל קישור מובנה ל-PayBox), שינוי הגדרות ונעילת הימורים.
- **נעילת טורניר מתקדמת:** אפשרות למנהל להחליט בזמן הנעילה האם להסיר משתתפים שלא שילמו או להשאיר אותם כצופים בלבד (עם 0 נקודות ואייקון 🚫).
- **לוח תוצאות וסטטיסטיקה:** לידרבורד ציבורי חי, כולל תובנות קבוצתיות (כמו מי הנבחרת הפייבוריטית בקבוצה).

## 🛠️ טכנולוגיות
- **Backend:** Node.js, Express
- **Frontend:** Vanilla JavaScript (ES6+), HTML5, CSS3 (Responsive Design)
- **Database:** JSON-based local database (`database.json`)

1. התקן תלויות: `npm install`
2. צור קובץ `.env` לפי `.env.example` והכנס ערכים אמיתיים
3. הרץ את השרת: `node server.js`
4. פתח בדפדפן: `http://localhost:3000`
